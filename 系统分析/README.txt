稳频仪上位机：STM32G431 频响与奈奎斯特稳定性分析软件
=================================================

稳频仪是一套基于 STM32G431 的嵌入式通用电路测量平台。
上位机负责把 STM32 扫频得到的输入/输出响应转换成伯德图、奈奎斯特图、
截止频率、增益裕度、相位裕度和稳定性判据结果。

这版仍然尽量和你给的 MATLAB 脚本处理链保持一致，同时强化奈奎斯特判据展示。

核心原则：
1. 串口读取方式按 MATLAB：严格读三行
   omega=[...]
   Magnitude_data=[...]
   Phase_data_rad=[...]

2. 默认不做 G431 频率轴修正
   这样和你 MATLAB 直接读串口文本时的行为一致。

3. 默认对幅值和相位做与 MATLAB 相同的平滑
   优先使用 Savitzky-Golay；若无 scipy，则退化为移动平均。

4. 截止频率、奈奎斯特图、伯德图、稳定性分析都按 MATLAB 脚本复现。

5. 奈奎斯特判据支持输入开环右半平面极点数 P。
   报告会显示：
   N = Nyquist 顺时针包围数
   P = 开环右半平面极点数
   Z = P + N = 闭环右半平面极点数
   当 Z = 0 时，按当前数据和 P 设置判定闭环稳定。

建议：
- 先关闭旧版“G431 兼容增强版”再运行这版。
- 先用“导入文本测试”把你 MATLAB 使用的同一组文本喂进去，确认图形一致。
- 确认一致后，再接串口实时读取。
- 参加比赛演示时，建议先用 RC 低通展示 -3 dB 截止频率，再切到奈奎斯特图讲 N/P/Z 稳定性判据。
