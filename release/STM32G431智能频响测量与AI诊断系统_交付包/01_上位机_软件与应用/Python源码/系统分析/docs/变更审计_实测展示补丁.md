这次变更只作为演示图小补丁，不作为正式底层分析改造。主程序仍以实测数据为准：

- `omega`
- `Magnitude_data`
- `Phase_data_rad`

- 主程序入口保留在原位置：
  - `start_app.bat`
  - `start_ui.bat`
  - `main.py`

- `build/`、`dist/`、`__pycache__/` 等构建/缓存产物已移到 `_artifacts/`。
